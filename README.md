React iOS Switch
===
[React switch component with a draggable button](http://clariussystems.github.io/react-ios-switch)

```javascript
import 'react-ios-switch/index.css'
import Switch from 'react-ios-switch'

<Switch
  checked={true}
  onChange={checked => ...}/>
```

Local Development
---
```
npm install
npm start
# Visit http://localhost:8080
```
